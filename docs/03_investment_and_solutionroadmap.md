# 03. Investment & Solution Roadmap View

## 視圖概述
此視圖將高階的能力需求轉化為具體的實施計畫（工作包），以評估資源投入與效益。

1. 能力實現：展示 CISO 規劃的安全能力（如 Data Protection）如何透過年度工作包（如 DLP/EDR/XDR 部署）來具體實現，作為預算分配和里程碑追蹤的基礎。

2. 技術依賴：說明如 DevSecOps 這樣的作業包，如何依賴底層的容器安全 (Container Security) 和 API 安全等技術服務來達成目標。

3. 關鍵組件價值：強調 SOC/SOAR 系統作為關鍵應用組件，如何直接服務於提升事件應變 (Incident Response) 的效率，從而證明對這些系統投資的必要性。
